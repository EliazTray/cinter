> cinter can fix `style` in the files which are endswith `.vue`, `.scss`, `.css`, `.less`.

## Install

```bash
yarn add cinter -g
or 
npm i cinter -g
```

## Usage

The tool will fix the file endwith `vue`, `scss`, `css`, `less` in `src` directory.

```bash
cinter fix 'src/**/*.{scss,css,less,vue}‘
```
